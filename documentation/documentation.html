AutoTalsi – HTML Website Template

AutoTalsi is a clean and modern HTML template designed for car services, auto repair shops, car dealers and automotive-related businesses. The template is built with HTML5, CSS3 and JavaScript and is fully responsive.

This documentation will help you to install, customize and manage the template.

Files Structure

autoTalsi/
│
├── app/
│   ├── css/
│   ├── js/
│   ├── images/
│   ├── fonts/
│   └── *.html
│
├── documentation.html
└── README.txt

Installation

  1. Unzip the downloaded archive.
  2. Open the app folder.
  3. Locate the index.html file.
  4. Open it in your browser or upload all files to your hosting server.

The template does not require any server-side configuration and works out of the box.

Editing Content

All content is located inside HTML files. You can edit text using any code editor (VS Code, Sublime Text, Notepad++).

To change text, simply edit the content between HTML tags.

Editing Images

All images are located in the app/images folder.

Replace images with your own while keeping the same file names or update image paths inside HTML files.

Editing Styles

CSS files are located in the app/css folder.

You can change colors, fonts and layout styles inside these files.

Updating the Template

To update the template, replace the old files with the new version files. We recommend backing up your current project before updating.

If you modified HTML or CSS files, merge your changes carefully.

Fonts & Icons

The template uses Google Fonts. Font links are included in the HTML <head> section.

Icons are implemented via icon font or SVG files.
